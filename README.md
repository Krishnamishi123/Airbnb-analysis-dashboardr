# Airbnb Global Performance Dashboard

## 📌 Project Overview

This project is an end-to-end Business Intelligence solution built in Power BI to analyze Airbnb marketplace performance across multiple cities worldwide.

The dashboard transforms raw Airbnb Listings and Reviews data into actionable business insights by tracking marketplace growth, customer satisfaction, pricing trends, review behavior, trust indicators, and city-level performance.

The solution demonstrates advanced Power BI capabilities including:

- Data Modeling
- DAX Calculations
- KPI Development
- Pareto Analysis
- Heatmaps
- Interactive Bookmarks
- Drill-through Navigation
- Business Storytelling

---

## 🎯 Business Objective

Airbnb operates in a highly competitive marketplace where:

- Hosts influence supply growth
- Customer reviews affect trust
- Pricing impacts profitability
- Guest satisfaction drives retention

The objective of this dashboard is to provide stakeholders with a centralized reporting platform to:

- Monitor marketplace expansion
- Track listing inventory
- Evaluate host ecosystem performance
- Analyze customer behavior
- Identify top-performing cities
- Improve strategic decision-making

---

## 📊 Dashboard Pages

### 1. Overview Dashboard

Provides a high-level view of Airbnb's global marketplace performance.

#### Key KPIs

- Total Listings
- Total Hosts
- Total Reviews
- Total Cities
- Property Types

#### Key Insights

- Airbnb experienced rapid growth between 2011–2016.
- Listing volume peaked during the maturity stage.
- Growth slowed due to regulatory restrictions.
- COVID-19 significantly impacted listing activity.

#### Visuals Used

- KPI Cards
- Multi-line Trend Analysis
- Lifecycle Growth Visualization
- Dynamic Narratives

---

### 2. Ratings Dashboard

Focuses on customer satisfaction and city-level quality analysis.

#### Key Analyses

##### Market Share Analysis

Identifies cities contributing the largest share of listings.

##### Pricing Analysis

Compares average prices across room categories:

- Hotel Room
- Entire Place
- Private Room
- Shared Room

##### Ratings Heatmap

Compares:

- Accuracy
- Cleanliness
- Communication
- Location
- Value

#### Key Insights

- Paris, NYC, and Sydney contribute a significant share of listings.
- Hotel Rooms command premium pricing.
- Mexico City and Rio de Janeiro show the highest customer ratings.
- Hong Kong and Istanbul present improvement opportunities.

#### Advanced Features

- Bookmark Navigation
- Dynamic View Switching
- Interactive Rating Analysis

---

### 3. Reviews Dashboard

Analyzes customer engagement, seasonality, and trust.

#### Key Analyses

##### Review Frequency

Measures customer review behavior.

##### Seasonality Analysis

Tracks review activity by month and city.

##### Trust Metrics

Analyzes:

- Identity Verification
- Profile Picture Adoption
- Host Trust Signals

#### Key Insights

- Most customers leave only one review.
- Review activity follows seasonal travel patterns.
- More than two-thirds of hosts are fully verified.

---

## 🛠 Data Model

The project follows a Star Schema approach.

### Fact Tables

#### Listings

Contains:

- Listing Information
- Pricing
- Room Type
- Property Type
- Host Information

#### Reviews

Contains:

- Review Counts
- Review Dates
- Customer Feedback Metrics

### Dimensions

- City
- Host
- Property Type
- Room Type
